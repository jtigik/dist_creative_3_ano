# Plano de Trabalho Docente – 2º Semestre de 2025

**Curso**: Técnico em Informática – Ensino Médio Profissionalizante  
**Disciplina**: Java Mobile  
**Unidade**: Senac Distrito Criativo, Porto Alegre/RS  
**Docente**: Jeferson Tigik  
**Carga Horária**: 65 horas-aula (13 semanas, ~5 horas-aula semanais)  
**Período**: Julho a Outubro de 2025  
**Público-Alvo**: Alunos do 3º ano do Ensino Médio Integrado ao Técnico em Informática  

## 1. Justificativa
A disciplina **Java Mobile** integra o eixo tecnológico do curso Técnico em Informática, alinhando-se à proposta do Senac de formar profissionais com competências técnicas e socioemocionais para o mercado de tecnologia. O foco no desenvolvimento de aplicações móveis com **Java** e **Android Studio** capacita os alunos a criar aplicativos Android funcionais, atendendo à demanda por soluções móveis. A disciplina promove o pensamento computacional, a resolução de problemas, a criatividade e a colaboração, conectando os conteúdos às competências da BNCC (pensamento científico, crítico e criativo) e preparando os alunos do 3º ano para o mercado de TI, estágios, ENEM e vestibulares. A abordagem baseada em projetos, conforme o PPP, reforça a interdisciplinaridade e a conexão com o contexto do Distrito Criativo.

## 2. Objetivos

### Objetivo Geral
Capacitar os alunos do 3º ano do Ensino Médio Técnico em Informática a desenvolver aplicações móveis utilizando Java e Android Studio, integrando competências de programação, design de interfaces e resolução de problemas para atuar no mercado de tecnologia móvel.

### Objetivos Específicos
1. Compreender os fundamentos de programação em Java aplicados ao desenvolvimento mobile.  
2. Dominar o Android Studio para criar aplicativos Android funcionais.  
3. Desenvolver interfaces de usuário (UI) para aplicativos móveis com XML e Java.  
4. Implementar projetos práticos que integrem programação e boas práticas de desenvolvimento.  
5. Fomentar habilidades socioemocionais, como trabalho em equipe, comunicação e pensamento crítico, por meio de metodologias ativas.

## 3. Conteúdos Programáticos
Os conteúdos foram organizados em 13 semanas (65 horas-aula), com foco em fundamentos de Java, desenvolvimento Android e um Projeto Integrador, ajustados para a carga horária reduzida.

| Semana | Conteúdo | Habilidades/Competências | Carga Horária |
|--------|----------|--------------------------|---------------|
| **1** (Julho) | **Fundamentos de Java para Mobile** <br> - Revisão de Java: sintaxe, classes, objetos <br> - Introdução ao Android Studio: configuração, emuladores | Compreender Java orientado a objetos; configurar o Android Studio. | 5h |
| **2-3** (Julho) | **Interfaces Android com XML** <br> - Layouts XML (LinearLayout, ConstraintLayout) <br> - Componentes de UI (TextView, Button, EditText) <br> - Event listeners em Java | Criar interfaces básicas; implementar interações com eventos. | 10h |
| **4-5** (Agosto) | **Lógica de Programação Mobile** <br> - Intents: navegação entre Activities <br> - Manipulação de dados (SharedPreferences, Listas) | Desenvolver lógica para navegação e armazenamento de dados. | 10h |
| **6-7** (Agosto) | **Persistência de Dados** <br> - Banco de dados SQLite <br> - CRUD (Create, Read, Update, Delete) em Java | Implementar persistência local; realizar operações CRUD. | 10h |
| **8-9** (Setembro) | **Integração com APIs e Testes** <br> - Consumo de APIs REST com Retrofit <br> - Testes básicos de funcionalidade | Integrar aplicativos com APIs externas; realizar testes. | 10h |
| **10-11** (Setembro) | **Design e Boas Práticas** <br> - Princípios de Material Design <br> - Acessibilidade em aplicativos móveis | Criar interfaces acessíveis; aplicar boas práticas de design. | 10h |
| **12-13** (Outubro) | **Projeto Integrador** <br> - Desenvolvimento de um aplicativo Android (planejamento, implementação, testes) <br> - Apresentação com pitch profissional | Integrar Java, XML e boas práticas; apresentar soluções criativas. | 10h |

## 4. Metodologias de Ensino
Alinhadas ao PPP do Senac, as metodologias são ativas e interdisciplinares:
- **Aprendizagem Baseada em Projetos (PBL)**: Desenvolvimento de um **Projeto Integrador**, criando um aplicativo Android (ex.: app para uma iniciativa do Distrito Criativo, como gestão de eventos comunitários).  
- **Gamificação**: Desafios semanais, como criar telas ou funcionalidades em tempo limitado.  
- **Estudos de Caso**: Análise de aplicativos reais (ex.: apps de delivery) para discutir arquitetura e usabilidade.  
- **Trabalho Colaborativo**: Uso de GitHub para versionamento e colaboração em projetos.  
- **Oficinas Práticas**: Sessões no laboratório de informática para prática no Android Studio.  
- **Palestras e Visitas Técnicas**: Convite a desenvolvedores mobile ou visitas a empresas no Distrito Criativo.

## 5. Recursos Didáticos
- **Tecnológicos**: Computadores com Android Studio, emuladores Android, GitHub, Postman (para testes de API).  
- **Materiais**: Slides, tutoriais em vídeo, exemplos de código Java/XML, guias de Material Design e acessibilidade.  
- **Ambientes**: Laboratório de informática, salas interativas e espaços colaborativos do Senac Distrito Criativo.  
- **Convidados**: Desenvolvedores Android para palestras ou mentorias.

## 6. Avaliação da Aprendizagem
A avaliação segue o **Modelo Pedagógico Senac**, com foco em competências:
- **Formativa**: Participação em atividades, exercícios práticos (ex.: telas XML, funções Java) e dinâmicas em grupo (30% da nota).  
- **Somativa**: Entrega de funcionalidades e mini-aplicativos no Android Studio, com feedback individualizado (30% da nota).  
- **Projeto Integrador**: Desenvolvimento e apresentação de um aplicativo Android, avaliado por funcionalidade, usabilidade, acessibilidade, qualidade do código e apresentação (40% da nota).  
- **Critérios de Avaliação**:
  - Domínio técnico de Java e Android Studio.  
  - Aplicação de boas práticas de programação e design.  
  - Qualidade funcional e usabilidade do aplicativo.  
  - Colaboração, comunicação e pensamento crítico.  
- **Recuperação Contínua**: Atividades personalizadas para reforçar conteúdos, com base na Avaliação Inteligente.

## 7. Integração com o Projeto de Vida e Projeto Profissional
- **Projeto de Vida**: Reflexões sobre o impacto do desenvolvimento mobile na sociedade e alinhamento com interesses pessoais na carreira em TI.  
- **Projeto Profissional**: Criação de portfólios com aplicativos desenvolvidos, simulações de entrevistas com profissionais de TI e preparação para estágios.

## 9. Resultados Esperados
Ao final do semestre, espera-se que os alunos:
- Desenvolvam aplicativos Android funcionais com Java e Android Studio.  
- Apliquem boas práticas de programação, design e acessibilidade.  
- Integrem lógica, interfaces e APIs em projetos práticos.  
- Desenvolvam habilidades de colaboração, comunicação e pensamento crítico.  
- Estejam preparados para estágios em TI, com portfólios de aplicativos.  
- Relacionem os conteúdos com as competências da BNCC, como pensamento computacional e resolução de problemas.
