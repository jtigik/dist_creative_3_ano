# Plano de Trabalho Docente – 2º Semestre de 2025

**Curso**: Técnico em Informática – Ensino Médio Profissionalizante  
**Disciplina**: Java Mobile  
**Unidade**: Senac Distrito Criativo, Porto Alegre/RS  
**Docente**: Jeferson Tigik  
**Carga Horária**: 120 horas-aula (20 semanas, 6 horas-aula semanais, 1 hora-aula = 50 minutos)  
**Período**: Julho a Dezembro de 2025  
**Público-Alvo**: Alunos do 3º ano do Ensino Médio Integrado ao Técnico em Informática  

## 1. Justificativa
A disciplina **Java Mobile** integra o eixo tecnológico do curso Técnico em Informática, alinhando-se à proposta do Senac de formar profissionais com competências técnicas e socioemocionais para o mercado de tecnologia. O foco no desenvolvimento de aplicações móveis com **Java** e **Android Studio** capacita os alunos a criar aplicativos Android funcionais, atendendo à crescente demanda por soluções móveis. A disciplina promove o pensamento computacional, a resolução de problemas, a criatividade e a colaboração, conectando os conteúdos às competências da BNCC, como pensamento científico, crítico e criativo, e preparando os alunos para o mercado de TI, estágios, ENEM e vestibulares. A abordagem baseada em projetos, conforme o PPP, reforça a interdisciplinaridade e a conexão com o contexto do Distrito Criativo.

## 2. Objetivos

### Objetivo Geral
Capacitar os alunos do 3º ano do Ensino Médio Técnico em Informática a desenvolver aplicações móveis utilizando Java e Android Studio, integrando competências de programação, design de interfaces e resolução de problemas para atuar no mercado de tecnologia móvel.

### Objetivos Específicos
1. Compreender os fundamentos de programação em Java aplicados ao desenvolvimento mobile.  
2. Dominar o ambiente Android Studio para criar aplicativos Android funcionais.  
3. Desenvolver interfaces de usuário (UI) para aplicativos móveis com XML e Java.  
4. Implementar projetos práticos que integrem programação, testes e boas práticas de desenvolvimento.  
5. Fomentar habilidades socioemocionais, como trabalho em equipe, comunicação e pensamento crítico, por meio de metodologias ativas.  

## 3. Conteúdos Programáticos
Os conteúdos foram organizados em 20 semanas (120 horas-aula), com ênfase em programação Java, desenvolvimento Android e projetos práticos, divididos em módulos progressivos.

| Semana | Conteúdo | Habilidades/Competências | Carga Horária |
|--------|----------|--------------------------|---------------|
| **1-2** (Julho) | **Fundamentos de Java para Mobile** <br> - Revisão de Java: sintaxe, classes, objetos, herança <br> - Introdução ao Android Studio: configuração, emuladores, estrutura de projetos | Compreender Java orientado a objetos; configurar e explorar o Android Studio. | 12h |
| **3-4** (Julho) | **Interfaces Android com XML** <br> - Estrutura de layouts XML (LinearLayout, ConstraintLayout) <br> - Componentes de UI (TextView, Button, EditText) <br> - Event listeners em Java | Criar interfaces básicas; implementar interações com eventos. | 12h |
| **5-6** (Agosto) | **Lógica de Programação Mobile** <br> - Manipulação de dados (Listas, Arrays, SharedPreferences) <br> - Intents: navegação entre Activities | Desenvolver lógica para navegação e armazenamento de dados. | 12h |
| **7-8** (Agosto) | **Persistência de Dados** <br> - Banco de dados SQLite <br> - CRUD (Create, Read, Update, Delete) em Java | Implementar persistência local de dados; realizar operações CRUD. | 12h |
| **9-10** (Setembro) | **Integração com APIs e Testes** <br> - Consumo de APIs REST com Retrofit <br> - Testes unitários e de interface com JUnit | Integrar aplicativos com APIs externas; realizar testes básicos. | 12h |
| **11-12** (Setembro) | **Design e Boas Práticas** <br> - Padrões de design (MVC, MVVM) <br> - Princípios de Material Design <br> - Acessibilidade em aplicativos móveis | Aplicar padrões de design e boas práticas; criar interfaces acessíveis. | 12h |
| **13-14** (Outubro) | **Gerenciamento de Projetos Android** <br> - Uso de Git para versionamento <br> - Publicação de aplicativos (Google Play Console) | Gerenciar projetos com Git; compreender o processo de publicação. | 12h |
| **15-16** (Outubro) | **Projeto Integrador – Parte 1** <br> - Definição do problema e planejamento do aplicativo <br> - Prototipagem da interface e lógica inicial | Planejar um aplicativo; iniciar desenvolvimento com Java e XML. | 12h |
| **17-18** (Novembro) | **Projeto Integrador – Parte 2** <br> - Implementação de funcionalidades (CRUD, APIs, navegação) <br> - Testes de usabilidade e ajustes | Implementar e testar o aplicativo; iterar com base em feedback. | 12h |
| **19-20** (Dezembro) | **Projeto Integrador – Finalização** <br> - Finalização do aplicativo <br> - Apresentação com pitch profissional | Concluir o aplicativo; apresentar soluções criativas; colaborar em equipe. | 12h |

## 4. Metodologias de Ensino
Alinhadas ao PPP do Senac, as metodologias são ativas, interdisciplinares e centradas em projetos:
- **Aprendizagem Baseada em Projetos (PBL)**: Desenvolvimento de um **Projeto Integrador**, criando um aplicativo Android funcional (ex.: app para uma iniciativa do Distrito Criativo, como gestão de eventos).  
- **Gamificação**: Desafios semanais, como criar funcionalidades em tempo limitado (ex.: tela de login, integração com API).  
- **Estudos de Caso**: Análise de aplicativos reais (ex.: apps de delivery ou redes sociais) para discutir arquitetura e usabilidade.  
- **Trabalho Colaborativo**: Uso de GitHub para versionamento e colaboração em projetos.  
- **Oficinas Práticas**: Sessões no laboratório de informática para prática no Android Studio.  
- **Palestras e Visitas Técnicas**: Convite a desenvolvedores mobile ou visitas a empresas de tecnologia no Distrito Criativo.

## 5. Recursos Didáticos
- **Tecnológicos**: Computadores com Android Studio, emuladores Android, GitHub, Postman (para testes de API).  
- **Materiais**: Slides, tutoriais em vídeo, exemplos de código Java/XML, guias de Material Design e acessibilidade.  
- **Ambientes**: Laboratório de informática, salas interativas e espaços colaborativos do Senac Distrito Criativo.  
- **Convidados**: Desenvolvedores Android para palestras ou mentorias.

## 6. Avaliação da Aprendizagem
A avaliação segue o **Modelo Pedagógico Senac**, com foco em competências:
- **Formativa**: Participação em atividades, exercícios práticos (ex.: telas XML, funções Java) e dinâmicas em grupo (30% da nota).  
- **Somativa**: Entrega de funcionalidades e mini-aplicativos no Android Studio, com feedback individualizado (30% da nota).  
- **Projeto Integrador**: Desenvolvimento e apresentação de um aplicativo Android, avaliado por funcionalidade, usabilidade, acessibilidade, qualidade do código e apresentação (40% da nota).  
- **Critérios de Avaliação**:
  - Domínio técnico de Java e Android Studio.
  - Aplicação de boas práticas de programação e design.
  - Qualidade funcional e usabilidade do aplicativo.
  - Colaboração, comunicação e pensamento crítico.
- **Recuperação Contínua**: Atividades personalizadas para reforçar conteúdos, com base na Avaliação Inteligente.

## 7. Integração com o Projeto de Vida e Projeto Profissional
- **Projeto de Vida**: Reflexões sobre o impacto do desenvolvimento mobile na sociedade e como os alunos podem alinhar seus interesses à carreira em TI.  
- **Projeto Profissional**: Criação de portfólios com os aplicativos desenvolvidos, simulações de entrevistas com profissionais de TI e preparação para estágios.

- Desenvolvam aplicativos Android funcionais com Java e Android Studio.

- Relacionem os conteúdos com as competências da BNCC, como pensamento computacional e resolução de problemas.
